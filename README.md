# Real-Time Chat Application

A simple real-time chat application built using **Flask** and **Flask-SocketIO**.  
Users can send and receive messages instantly with live updates.

## Features
- Real-time two-way messaging
- WebSocket-based communication
- Clean and responsive UI
- Lightweight Flask backend

## Tech Stack
- Python (Flask)
- Flask-SocketIO
- HTML, CSS
- JavaScript

## How to Run
1. Install dependencies  
   ```bash
    pip install flask flask-socketio
2. Run the app
     python app.py
3. Open browser
    http://127.0.0.1:5000
